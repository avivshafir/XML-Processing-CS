XML Basic Processing Samples
--------------------------------------------------------
If you wish to test out the Write from Database sample, you will need to download the AdventureWorksLT database
Open the App.Config file and do the following:
  Modify the connection string to point to your server and the AdventureWorksLT database
  Modify the 'XmlFile' and 'XsdFile' paths
